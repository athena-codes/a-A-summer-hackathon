# How to setup backend

## Prepare your env file
change directory to app/backend
1. cp .env.example to .env
2. put your gemini api key to .env

## Install node modules
change directory to app/backend
1. npm install

## Ensure Environment set up
Using Node V18.20.4
npm V10.7.0

## Start
change directory to app/backend
1. node server
